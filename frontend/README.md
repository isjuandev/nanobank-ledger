# Frontend (Angular)

Aplicación Angular basada en Standalone Components + Signals para gestión de billeteras y transacciones.

## Development server

```bash
npm start
```

## Build

```bash
npm run build
```

## Unit testing (Jest)

```bash
npm test
```

## Coverage (Jest)

```bash
npm run test:coverage
```

Umbral obligatorio: **Line Coverage >= 80%** (configurado en `jest.config.cjs`).
